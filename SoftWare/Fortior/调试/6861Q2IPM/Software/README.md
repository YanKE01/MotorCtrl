# Standard Software for compressor

> 这是基于FU6861Q2的单电阻无感FOC标准压缩机程序  
> 如有BUG或者优化建议，可提交到标准程序对应的议题

## 适用编译器


- [支持] KeilC51
- [不支持] FTCC

## 适用芯片
- FU6861Q2

## 发布记录


V1.0.0(2022-11-18)
- 初始版本
- 基于FU6861Q2的单电阻无感FOC标准压缩机程序
